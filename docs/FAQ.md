# OpenClaw 私有部署体检 FAQ

## 1. 这个产品和安全咨询有什么区别？
区别在于：这不是先卖大量人工咨询，而是先卖一套更轻、更标准化的体检结果。
你先拿到风险评分、问题分组、修复优先级和基线方向，再决定是否升级。

## 2. 它适合什么人？
适合已经自托管 OpenClaw、准备接入真实账号/节点/浏览器/消息渠道，或者准备给客户做交付的人。

## 3. 它不适合什么人？
不太适合完全没部署过 OpenClaw、还停留在围观阶段的人；也不适合作为重度企业合规项目的一步到位替代品。

## 4. 免费预检能拿到什么？
你会拿到基础风险预览、初步评分，以及是否建议升级正式体检的判断。

## 5. 标准版和 Pro 修复版有什么区别？
标准版更偏“看清问题、确认风险、拿正式报告”；
Pro 修复版更偏“拿修复动作清单、基线建议、日志/巡检/告警方向”。

## 6. 为什么不直接做持续监控？
持续监控是更长期的升级方向，但第一步更容易成交的是“先判断当前部署能不能安全上线”。

## 7. 这份报告最适合拿来做什么？
最适合：
- 上线前检查
- 内部汇报
- 给客户或老板说明风险状态
- 作为后续整改和巡检的起点

## 8. 会检查哪些典型问题？
重点覆盖：
- 网关和公网暴露
- 权限过宽
- 日志 / 审计缺失
- 节点 / 浏览器能力风险
- Skill 来源和调用风险
- 成本和资源监控缺失

## 9. 为什么值得付费，而不是自己看清单？
因为真正值钱的不只是“列问题”，而是：
- 风险分级
- 修复优先级
- 更贴 OpenClaw 场景的检查规则
- 更像交付物的报告结构

## 10. 这个产品后续会怎么升级？
先从体检和修复版开始，后续再升级到持续巡检、团队版和更完整的运营/告警层。
