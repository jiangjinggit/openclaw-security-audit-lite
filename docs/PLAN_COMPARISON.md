# OpenClaw 套餐对比表

| 套餐 | 适合谁 | 主要交付物 | 价格 |
|---|---|---|---|
| 免费预检 | 想先判断问题严不严重的人 | 风险预览、初步评分、是否建议升级 | 免费 |
| 299 元体检报告 | 想正式判断能否上线的人 | 风险评分、风险分组、整改优先级、正式报告、轻量运维观测建议 | 299 元起 |
| Pro 修复版 | 想直接知道下一步怎么修的人 | 体检报告全部内容 + 修复动作清单 + 基线建议 + 日志/告警方向 | 699-1999 元 |
| 团队 / 企业版 | 多实例、交付型、团队试点场景 | 多实例汇总、归档、权限差异、团队/交付输出 | 5000 元+ |

## 当前推荐顺序
免费预检 → 299 元体检报告 → Pro 修复版 → 团队 / 企业版

## 设计原则
- 免费预检负责降低进入门槛
- 299 元体检报告负责验证首单
- Pro 负责提高客单和解决深度
- 团队版负责承接更复杂场景
