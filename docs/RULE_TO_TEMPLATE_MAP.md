# 规则库到模板包映射

## 当前作用
这份映射用于把“发现问题”直接连到“套哪类模板先修”。

---

## 当前首版映射方向
1. 暴露面问题 → 反向代理 / 来源限制 / 网关前置模板
2. 鉴权问题 → 网关鉴权 / 入口保护模板
3. 权限边界问题 → systemd / 运行权限拆分模板
4. Skill 来源问题 → skill 白名单 / 上线前检查模板
5. 审计留痕问题 → 日志保留 / 审计记录模板
6. 成本与巡检问题 → 复检与预算波动检查模板

---

## 当前目标
让报告输出不止停留在“建议”，而是能继续指向：
- 哪类模板最适合先套
- 哪个修复动作最该先做
- 哪些问题进入复检而不是当天立刻处理
