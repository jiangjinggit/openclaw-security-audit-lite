# OpenClaw 安全审计与权限基线包

## 定位
这是 `openclaw-security-audit-lite` 当前最适合作为首单变现的产品化包装。

它卖的不是泛安全概念，而是一个明确结果：

> 帮用户在 OpenClaw 上线前，看清风险、排出优先级、拿到可执行整改建议。

## 适合谁
- 已经部署 OpenClaw 的独立开发者
- 帮客户做 Agent / OpenClaw 私有化交付的技术服务商
- 准备在团队内部试点 OpenClaw 的技术负责人
- 接入了 Telegram / Feishu / WhatsApp / Browser / Node 的重度用户

## 解决什么问题
- 不知道当前部署是否适合进入真实环境
- 不知道哪些权限过宽、哪些暴露面最危险
- 不知道日志、节点、消息通道、技能来源是否留下风险
- 不知道应该先修什么，缺少整改优先级
- 对内汇报时拿不出清晰、标准化的风险说明

## 最小可售卖版本
### 1. 标准报告版
交付物：
- 风险评分
- 高/中/低风险分组
- 问题说明
- 整改优先级
- Markdown/PDF 报告

建议价格：`199-499 元`

### 2. 整改升级版
交付物：
- 标准报告版全部内容
- 基线配置建议
- 权限收敛建议
- 日志与观测建议
- 修复动作清单

建议价格：`699-1999 元`

### 3. 团队/企业版
交付物：
- 多节点/多环境巡检
- 历史报告归档
- 权限差异对比
- 审计输出模板
- 私有化部署增强说明

建议价格：`5000 元+`

## 为什么适合作为首单
1. 付费理由清晰：降低风险、少踩坑、敢上线
2. 结果明确：报告、分级、建议、模板
3. 异步交付：无需高频会议
4. 可标准化：便于后续沉淀规则库与模板

## 和三项目矩阵的关系
- `openclaw-security-audit-lite`：首单切口 / 风险报告 / 权限基线
- `openclaw-monitor-lite`：后续升级层 / 监控 / 告警 / 成本 / 观测
- `openclaw-template-market`：增长入口 / 模板场景 / 低价前端产品
